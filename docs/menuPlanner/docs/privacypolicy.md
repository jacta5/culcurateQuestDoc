The App may collect and use the following types of information:

### (1) Firebase Authentication

- Apple account information (when logging in with Apple)

This information is used solely for authentication and will not be shared with any third parties.

User-generated content, such as recipes created or saved within the App, is stored in **Firebase Firestore**.  
This data is used only to provide App functionality and is not shared externally.
